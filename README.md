## Dropdown Menu

Watch the full [React dropdown tutorial](https://youtu.be/IF6k0uZuypA) on YouTube. 

Site is deployed on:
https://multilevel-dropdown-menu-with-react.netlify.app/

```
git clone <this-repo>
```
